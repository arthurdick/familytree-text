What is FTT?

FamilyTree-Text (FTT) is a proposed standard for storing genealogical data. Unlike GEDCOM (which can be verbose) or database blobs (which are opaque), FTT is designed to be written and read like a simple text document.

Core Principles

  • Human Readability: You can open an .ftt file in Notepad and understand it without software.
  • Git Friendliness: The line-based structure allows for clean "diffs," making it perfect for version-controlling family history.
  • Data Durability: No binary encoding or complex XML schemas. Just UTF-8 text.
